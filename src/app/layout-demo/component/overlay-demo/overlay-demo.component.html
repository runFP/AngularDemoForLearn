overlay-demo

overlay提供一种方法在屏幕中打开浮动面板

知识点
  • overlay依赖一小组结构样式才能正常工作，因此如果你没有使用Angular Material,需要引入@import '~@angular/cdk/overlay-prebuilt.css';
  • 可以通过componentRef.instance来手动为组件属性赋值，eg: Object.assign(componentRef.instance, properties)
  • 留意这个demo控制台输出的信息，注意组件形式的portal的组件生命周期执行顺序,对了解Angular的渲染机制非常有帮助
